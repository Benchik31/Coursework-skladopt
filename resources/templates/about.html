Стек и архитектура
  • Java 17, Spring Boot 3 (Web, Data JPA, Security), Thymeleaf.
  • PostgreSQL, ORM на Hibernate, репозитории Spring Data.
  • Слоистая архитектура: Controller → Service → Repository → DB.
  • Валидация через jakarta.validation, CSRF и разграничение ролей (ADMIN/USER).
  • UI: Bootstrap 5 + кастомная тема Fenix (градиенты #FA7355 / #6C48ED).
Что реализовано
  • Каталог товаров с карточками и фото, CRUD, фильтры.
  • Остатки по складам, резервы, движения (IN/OUT/TRANSFER).
  • Поставки (партии), клиенты/поставщики, склады, категории.
  • Заказы с жизненным циклом DRAFT → CONFIRMED → SHIPPED.
  • Отчёты: оборот, низкие остатки, подборки.
  • Админ-панель: пользователи, роли, справочники.
